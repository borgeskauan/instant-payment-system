# Atividades

Esta pasta guarda tarefas priorizadas. Cada tarefa tem seu próprio arquivo `.md`.

O status da tarefa é definido pela pasta:

- `agora/`: tarefa ativa.
- raiz de `Atividades/`: tarefa priorizada, mas pausada.

Os arquivos de tarefa devem manter o mesmo padrão interno para que seja possível mudar o status apenas movendo o arquivo de pasta, sem editar seu conteúdo.
